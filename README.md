# Telegram Cities Game Bot

Telegram-бот для игры в "Города". Бот использует API Telegram для взаимодействия с пользователем, GigaChat для проверки реальности городов и совершения ответных ходов, а также SpeechRecognition и pydub для обработки голосовых сообщений.

## Особенности

- Текстовая игра в классические "Города"
- Голосовой ввод (распознавание речи)
- Проверка названий городов через GigaChat
- Получение интересной информации об упомянутых городах
- Поддержка нескольких активных игр в разных чатах одновременно

## Требования

- Python 3.8+
- [FFmpeg](https://ffmpeg.org/) (необходим для работы `pydub`)

## Установка и запуск

1. Клонируйте репозиторий:
   ```bash
   git clone <URL_репозитория>
   cd tg_cities-game
   ```

2. Создайте виртуальное окружение и активируйте его:
   ```bash
   python3 -m venv .venv
   source .venv/bin/activate  # Для Linux/macOS
   # .venv\Scripts\activate   # Для Windows
   ```

3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```

4. Создайте файл `.env` на основе файла `env.example` и укажите в нём ваши токены:
   ```bash
   cp .env.example .env
   ```

5. Запустите бота:
   ```bash
   python gh.py
   ```

## Использование бота

- `/start` - начать новую игру
- `/stop` - завершить текущую игру
- `/status` - получить информацию о текущем состоянии игры (использованные города, чья очередь и т.д.)
- `/info` - получить информацию о последнем городе, названном ботом

Вы можете писать названия городов текстом или отправлять их голосовыми сообщениями — бот распознает их и сделает ответный ход.
